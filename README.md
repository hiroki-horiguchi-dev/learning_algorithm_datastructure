## 目的
- C 言語と無縁の生活を送ってきたので触れておきたい
- メモリ管理、値渡し、コマンドライン引数やファイルの入出力、構造体などを理解して簡単な RPG ゲームや双六プログラムみたいなものを書くまでを目標とする

- CS に詳しくなるべきだよな、という漠然とした思い

- Android エンジニア以外にも幅を増やせたら良い
